# AI Coding Agents Instructions

## 言語設定

すべてのAIコーディングエージェントは、以下の項目について**必ず日本語で回答・記述**してください。

- プルリクエストのタイトル
- プルリクエストの説明文
- コミットメッセージ
- コードレビューのコメント
- 作業内容の説明
- ユーザーへの返答・報告

## Language Settings

All AI coding agents must respond and write in **Japanese** for the following:

- Pull request titles
- Pull request descriptions
- Commit messages
- Code review comments
- Work explanations
- Responses and reports to users
